# TCP Socket (Client-server Model) using Winsock2

CMake was used as the build system generator of this application. Kindly follow these steps:
```
mkdir build
```
```
cd build
```
```
cmake ..
or
cmake .. -G "<generator_name>"
```
```
cmake --build .
```
```
cd bin
tcp-socket.exe
```
